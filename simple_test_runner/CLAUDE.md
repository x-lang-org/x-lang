# CLAUDE.md - simple_test_runner

## 功能定位

simple_test_runner 是 X 语言的轻量级测试运行工具，用于自动查找并运行测试文件，验证 X 语言实现的正确性。它是 X 语言测试基础设施的简化版本，专注于快速执行测试并报告结果。

### 与其他组件的关系

```
┌─────────────────────────────────────────────────────────────────────┐
│ simple_test_runner                                                 │
├─────────────────────────────────────────────────────────────────────┤
│ 依赖组件                                                           │
├─────────────────────────────────────────────────────────────────────┤
│ • x-lexer       : 词法分析器，将源代码转换为 token 流              │
│ • x-parser      : 语法分析器，将 token 流构建为 AST                │
│ • x-interpreter : 树遍历解释器，直接执行 AST                        │
│ • ../test/      : 测试文件存放目录（.x 文件）                       │
└─────────────────────────────────────────────────────────────────────┘
```

## 核心功能

1. **测试文件发现**：递归扫描测试目录，查找所有 .x 文件
2. **测试执行**：解析并执行每个测试文件
3. **结果统计**：统计通过和失败的测试数量
4. **进度显示**：实时显示测试进度和结果
5. **错误报告**：显示失败测试的错误信息

## 构建与运行

### 构建

```bash
cd /C/Users/Administrator/Documents/x-lang/simple_test_runner
cargo build
```

### 运行

```bash
cargo run
# 或者直接运行可执行文件
./target/debug/simple_test_runner
```

默认情况下，simple_test_runner 会扫描 `../test/` 目录下的所有 .x 文件。

## 源代码说明

### main.rs

main.rs 是 simple_test_runner 的唯一源代码文件，包含以下核心函数：

1. **main()**：主函数，协调测试流程
   - 发现测试文件
   - 遍历执行每个测试
   - 统计并显示最终结果

2. **find_test_files(dir)**：递归查找测试文件
   - 输入：目录路径
   - 输出：所有 .x 文件的路径列表
   - 支持递归扫描子目录

3. **run_test(file)**：执行单个测试文件
   - 读取文件内容
   - 调用 x-parser 解析为 AST
   - 调用 x-interpreter 执行
   - 返回成功或错误信息

## 当前状态与限制

### 已实现功能

- ✅ 递归测试文件发现
- ✅ 自动执行所有测试文件
- ✅ 逐个显示测试结果
- ✅ 统计通过和失败数量
- ✅ 错误信息报告
- ✅ 简洁明了的进度显示（✓/✗）

### 限制

- 不支持测试过滤（不能只运行特定测试）
- 不支持测试断言（通过标准是程序不崩溃）
- 不支持并行执行
- 不支持测试超时
- 缺少测试覆盖率统计
- 不支持测试跳过或标记
- 缺少测试结果持久化存储

## 开发流程

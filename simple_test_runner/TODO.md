# TODO.md - simple_test_runner

## 项目概述

simple_test_runner 是 X 语言的轻量级测试运行工具，用于自动查找并执行测试文件。当前实现已具备基本的测试发现和执行功能，但还需要增强以提供更全面的测试功能。

## 依赖关系

simple_test_runner 依赖以下内部组件，其变更会直接影响本项目：

- **x-lexer**：词法分析器
- **x-parser**：语法分析器
- **x-interpreter**：解释器
- **../test/**：测试文件目录

## 开发流程

### 常规开发流程

1. 修改 `src/main.rs`
2. 运行 `cargo run` 测试变更
3. 运行 `cargo test` 确保无错误
4. 提交更改

### 构建与运行

```bash
# 构建
cargo build

# 运行
cargo run

# 运行测试
cargo test

# 格式化代码
cargo fmt
```

## 测试用例示例

在 `../test/` 目录下添加测试文件时，可以考虑以下类型：

- **基础测试**：简单的变量和打印语句
- **函数测试**：函数定义和调用
- **控制流测试**：if/else、while 等控制结构
- **错误测试**：预期会产生错误的测试（需标记）

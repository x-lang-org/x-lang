# 不同的二叉搜索树 (Unique Binary Search Trees)

## 题目信息

- **难度**: MEDIUM
- **通过率**: 71.4%
- **标签**: `tree`, `binary-search-tree`, `math`, `dynamic-programming`, `binary-tree`
- **LeetCode**: https://leetcode.cn/problems/unique-binary-search-trees/

## 题目描述

Given an integer `n`, return <em>the number of structurally unique **BST&#39;**s (binary search trees) which has exactly </em>`n`<em> nodes of unique values from</em> `1` <em>to</em> `n`.


 

**Example 1:**
<img alt="" src="https://assets.leetcode.com/uploads/2021/01/18/uniquebstn3.jpg" style="width: 600px; height: 148px;" />
```

**Input:** n = 3
**Output:** 5

```


**Example 2:**

```

**Input:** n = 1
**Output:** 1

```


 

**Constraints:**



	- `1 &lt;= n &lt;= 19`

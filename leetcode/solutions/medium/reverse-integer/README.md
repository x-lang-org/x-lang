# 整数反转 (Reverse Integer)

## 题目信息

- **难度**: MEDIUM
- **通过率**: 35.6%
- **标签**: `math`
- **LeetCode**: https://leetcode.cn/problems/reverse-integer/

## 题目描述

Given a signed 32-bit integer `x`, return `x`<em> with its digits reversed</em>. If reversing `x` causes the value to go outside the signed 32-bit integer range `[-2<sup>31</sup>, 2<sup>31</sup> - 1]`, then return `0`.


**Assume the environment does not allow you to store 64-bit integers (signed or unsigned).**


 

**Example 1:**

```

**Input:** x = 123
**Output:** 321

```


**Example 2:**

```

**Input:** x = -123
**Output:** -321

```


**Example 3:**

```

**Input:** x = 120
**Output:** 21

```


 

**Constraints:**



	- `-2<sup>31</sup> &lt;= x &lt;= 2<sup>31</sup> - 1`

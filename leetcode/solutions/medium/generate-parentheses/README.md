# 括号生成 (Generate Parentheses)

## 题目信息

- **难度**: MEDIUM
- **通过率**: 79.1%
- **标签**: `string`, `dynamic-programming`, `backtracking`
- **LeetCode**: https://leetcode.cn/problems/generate-parentheses/

## 题目描述

Given `n` pairs of parentheses, write a function to <em>generate all combinations of well-formed parentheses</em>.


 

**Example 1:**
```
**Input:** n = 3
**Output:** ["((()))","(()())","(())()","()(())","()()()"]

```
**Example 2:**
```
**Input:** n = 1
**Output:** ["()"]

```

 

**Constraints:**



	- `1 &lt;= n &lt;= 8`

# lexer_test - X 语言词法分析器测试工具

本文件为 Claude Code (claude.ai/code) 提供在本子项目工作时的指导规范。

## 子项目概览

**lexer_test** 是一个简单的测试工具，用于演示和手动测试 `x-lexer` 词法分析器的功能。它读取固定的源代码字符串，调用词法分析器进行分词，并将结果打印到控制台。

### 功能定位

- 简单演示 x-lexer 的基本用法
- 手动测试词法分析器输出
- 开发调试辅助工具

## 当前状态

### ✅ 已实现

1. 基本框架：创建词法分析器实例
2. 简单分词：对 `"1 + 1"` 进行分词
3. 输出显示：打印 Token 和 Span 信息

### 🔴 待改进

1. 硬编码的源代码，无法从命令行参数或文件读取
2. 输出格式较为简单，不便于调试复杂代码
3. 无错误处理的详细展示
4. 无单元测试

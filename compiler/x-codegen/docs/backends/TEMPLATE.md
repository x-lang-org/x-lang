## 快速概览

### 后端的职责

一个后端需要：
1. **接收 LIR 模块** - 来自 `x-lir` crate 的低级中间表示
2. **翻译为目标格式** - 转换为目标语言源码、中间 IR 或字节码
3. **输出结果** - 返回可被目标工具链处理的内容
4. **集成编译工具** - 调用目标工具链完成最终编译（可选）

### 基本流程

```
LIR Module
    ↓
[后端特定的翻译器]
    ↓
目标格式
    ├─ 源码：存储为文件，供后续工具链处理
    ├─ IR：存储为标准格式
    └─ 字节码：直接输出或存储
    ↓
[目标工具链编译（可选）]
    ↓
可执行文件/库/脚本
```

## 第一阶段：源码翻译型后端

### 1.1 基本结构

所有源码翻译型后端都遵循相同的模式。以 TypeScript 为例：

```rust
// x-codegen/src/typescript_backend.rs

use x_lir::module::Module as LirModule;
use std::fmt::Write;

/// TypeScript 代码生成器
pub struct TypeScriptBackend;

impl TypeScriptBackend {
    /// 将 LIR 模块翻译为 TypeScript 源码
    pub fn codegen(module: &LirModule) -> Result<String, String> {
        let mut output = String::new();
        
        // 1. 生成文件头
        Self::write_header(&mut output)?;
        
        // 2. 生成全局变量
        Self::write_globals(&mut output, module)?;
        
        // 3. 生成类型定义
        Self::write_types(&mut output, module)?;
        
        // 4. 生成函数
        Self::write_functions(&mut output, module)?;
        
        Ok(output)
    }
    
    // ... 辅助方法
}
```

### 1.2 核心翻译方法

#### 1.2.1 生成文件头

```rust
fn write_header(output: &mut String) -> Result<(), String> {
    writeln!(output, "// Generated by X Compiler").map_err(|_| "write error")?;
    writeln!(output, "// DO NOT EDIT THIS FILE MANUALLY").map_err(|_| "write error")?;
    writeln!(output)?;
    
    // 导入必要的库
    writeln!(output, "// Runtime utilities")?;
    writeln!(output, "type Result<T> = [Error, T] | T;")?;
    
    Ok(())
}
```

#### 1.2.2 翻译全局变量

```rust
fn write_globals(output: &mut String, module: &LirModule) -> Result<(), String> {
    if module.globals.is_empty() {
        return Ok(());
    }
    
    writeln!(output, "// Global variables")?;
    for global in &module.globals {
        let type_str = Self::translate_type(&global.ty)?;
        writeln!(output, "let {}: {} = {};", 
                global.name, type_str, Self::translate_value(&global.init)?)?;
    }
    writeln!(output)?;
    
    Ok(())
}
```

#### 1.2.3 翻译函数

```rust
fn write_functions(output: &mut String, module: &LirModule) -> Result<(), String> {
    writeln!(output, "// Functions")?;
    for func in &module.functions {
        Self::write_function(output, func)?;
    }
    Ok(())
}

fn write_function(output: &mut String, func: &LirFunction) -> Result<(), String> {
    // 生成函数签名
    let params = func.params.iter()
        .map(|p| format!("{}: {}", p.name, Self::translate_type(&p.ty).unwrap_or_default()))
        .collect::<Vec<_>>()
        .join(", ");
    
    let return_type = Self::translate_type(&func.return_type)?;
    writeln!(output, "function {}({}) -> {} {{", func.name, params, return_type)?;
    
    // 生成函数体
    for block in &func.blocks {
        Self::write_basic_block(output, block)?;
    }
    
    writeln!(output, "}}")?;
    writeln!(output)?;
    
    Ok(())
}
```

#### 1.2.4 翻译基本块

```rust
fn write_basic_block(output: &mut String, block: &BasicBlock) -> Result<(), String> {
    writeln!(output, "  {}: {{", block.label)?;
    
    for instr in &block.instructions {
        Self::write_instruction(output, instr)?;
    }
    
    // 写入终结指令
    Self::write_terminator(output, &block.terminator)?;
    
    writeln!(output, "  }}")?;
    
    Ok(())
}
```

#### 1.2.5 翻译指令

```rust
fn write_instruction(output: &mut String, instr: &Instruction) -> Result<(), String> {
    match instr {
        Instruction::Assign { dest, value } => {
            let value_str = Self::translate_value(value)?;
            writeln!(output, "    {} = {};", dest, value_str)?;
        }
        Instruction::BinaryOp { dest, op, lhs, rhs } => {
            let op_str = match op {
                BinOp::Add => "+",
                BinOp::Sub => "-",
                BinOp::Mul => "*",
                BinOp::Div => "/",
                // ... 其他操作符
                _ => "?",
            };
            let lhs_str = Self::translate_value(lhs)?;
            let rhs_str = Self::translate_value(rhs)?;
            writeln!(output, "    {} = {} {} {};", dest, lhs_str, op_str, rhs_str)?;
        }
        Instruction::Call { dest, func, args } => {
            let args_str = args.iter()
                .map(|a| Self::translate_value(a))
                .collect::<Result<Vec<_>, _>>()?
                .join(", ");
            writeln!(output, "    {} = {}({});", dest, func, args_str)?;
        }
        // ... 其他指令类型
    }
    Ok(())
}
```

#### 1.2.6 翻译终结指令

```rust
fn write_terminator(output: &mut String, term: &Terminator) -> Result<(), String> {
    match term {
        Terminator::Return { value: Some(val) } => {
            let val_str = Self::translate_value(val)?;
            writeln!(output, "    return {};", val_str)?;
        }
        Terminator::Return { value: None } => {
            writeln!(output, "    return;")?;
        }
        Terminator::Branch { target } => {
            writeln!(output, "    goto {};", target)?;
        }
        Terminator::CondBranch { cond, then_bb, else_bb } => {
            let cond_str = Self::translate_value(cond)?;
            writeln!(output, "    if ({}) {{ goto {}; }} else {{ goto {}; }}", 
                    cond_str, then_bb, else_bb)?;
        }
    }
    Ok(())
}
```

#### 1.2.7 翻译值和类型

```rust
fn translate_value(value: &Value) -> Result<String, String> {
    match value {
        Value::Const(c) => Self::translate_const(c),
        Value::Local(name) => Ok(name.clone()),
        Value::Global(name) => Ok(name.clone()),
        Value::Param(name) => Ok(name.clone()),
        // ... 其他值类型
    }
}

fn translate_type(ty: &Type) -> Result<String, String> {
    match ty {
        Type::Int => Ok("number".to_string()),
        Type::Bool => Ok("boolean".to_string()),
        Type::String => Ok("string".to_string()),
        Type::Array(elem_ty) => {
            let elem = Self::translate_type(elem_ty)?;
            Ok(format!("{}[]", elem))
        }
        // ... 其他类型
    }
}

fn translate_const(c: &Const) -> Result<String, String> {
    match c {
        Const::Int(n) => Ok(n.to_string()),
        Const::Bool(b) => Ok(b.to_string()),
        Const::String(s) => Ok(format!("\"{}\"", s)),
        // ... 其他常数
    }
}
```

### 1.3 工具链集成

集成目标工具链（如 tsc）进行最终编译：

```rust
/// 调用 TypeScript 编译器编译生成的源码
pub fn compile_typescript(source_file: &str, output_file: &str) -> Result<(), String> {
    let output = std::process::Command::new("tsc")
        .arg(source_file)
        .arg("--outFile")
        .arg(output_file)
        .output()
        .map_err(|e| format!("Failed to run tsc: {}", e))?;
    
    if !output.status.success() {
        let stderr = String::from_utf8_lossy(&output.stderr);
        return Err(format!("tsc compilation failed:\n{}", stderr));
    }
    
    Ok(())
}

/// 端到端编译：X → TypeScript → JavaScript
pub fn compile(module: &LirModule, output_file: &str) -> Result<(), String> {
    // 生成 TypeScript 源码
    let ts_code = Self::codegen(module)?;
    
    // 写入临时文件
    let ts_file = format!("{}.ts", output_file);
    std::fs::write(&ts_file, &ts_code)
        .map_err(|e| format!("Failed to write TypeScript file: {}", e))?;
    
    // 调用 tsc 编译
    Self::compile_typescript(&ts_file, output_file)?;
    
    // 清理临时文件
    std::fs::remove_file(&ts_file).ok();
    
    Ok(())
}
```

### 1.4 完整示例

```rust
// x-codegen/src/typescript_backend.rs (完整版)

use x_lir::module::Module as LirModule;
use std::fmt::Write;

pub struct TypeScriptBackend;

impl TypeScriptBackend {
    pub fn codegen(module: &LirModule) -> Result<String, String> {
        let mut output = String::new();
        
        writeln!(output, "// Generated by X Compiler").map_err(|_| "write error")?;
        writeln!(output, "// DO NOT EDIT").map_err(|_| "write error")?;
        writeln!(output)?;
        
        // 写入全局变量
        for global in &module.globals {
            let ty = Self::type_to_ts(&global.ty)?;
            writeln!(output, "let {}: {} = 0;", global.name, ty)?;
        }
        
        // 写入函数
        for func in &module.functions {
            writeln!(output, "function {}() {{", func.name)?;
            for block in &func.blocks {
                writeln!(output, "  // Block {}", block.label)?;
                for instr in &block.instructions {
                    writeln!(output, "    // Instruction")?;
                }
            }
            writeln!(output, "}}")?;
        }
        
        Ok(output)
    }
    
    fn type_to_ts(ty: &str) -> Result<String, String> {
        match ty {
            "i32" | "i64" | "f32" | "f64" => Ok("number".to_string()),
            "bool" => Ok("boolean".to_string()),
            "str" => Ok("string".to_string()),
            _ => Err(format!("Unknown type: {}", ty)),
        }
    }
}
```

## 第一阶段：字节码翻译型后端

### 2.1 基本结构

字节码型后端（如 Erlang）的结构略有不同：

```rust
// x-codegen/src/erlang_backend.rs

use x_lir::module::Module as LirModule;

/// Erlang/BEAM 字节码生成器
pub struct ErlangBackend;

impl ErlangBackend {
    /// 将 LIR 直接翻译为 BEAM 字节码
    pub fn codegen(module: &LirModule) -> Result<Vec<u8>, String> {
        let mut bytecode = Vec::new();
        
        // 1. 写入 BEAM 文件头
        Self::write_beam_header(&mut bytecode)?;
        
        // 2. 编码原子表
        Self::write_atom_table(&mut bytecode, module)?;
        
        // 3. 编码代码段
        Self::write_code_section(&mut bytecode, module)?;
        
        // 4. 编码导出表
        Self::write_exports(&mut bytecode, module)?;
        
        // 计算总长度
        Self::finalize_beam(&mut bytecode)?;
        
        Ok(bytecode)
    }
    
    fn write_beam_header(bytecode: &mut Vec<u8>) -> Result<(), String> {
        // BEAM 文件必须以 "FOR1" 开头（大端 FORM 块）
        bytecode.extend_from_slice(b"FOR1");
        // 后续四个字节为大小（稍后填充）
        bytecode.extend_from_slice(&[0, 0, 0, 0]);
        bytecode.extend_from_slice(b"BEAM");
        Ok(())
    }
    
    // ... 其他方法
}
```

### 2.2 关键特性

字节码后端需要：
- 理解目标 VM 的指令集
- 处理运行时数据结构（如 BEAM 的术语格式）
- 正确编码字节码格式

### 4.1 处理错误

```rust
/// 结果类型
pub type CodegenResult<T> = Result<T, CodegenError>;

#[derive(Debug)]
pub struct CodegenError {
    pub message: String,
    pub location: Option<(usize, usize)>, // (line, column)
}

impl CodegenError {
    pub fn new(message: impl Into<String>) -> Self {
        Self {
            message: message.into(),
            location: None,
        }
    }
    
    pub fn with_location(mut self, line: usize, col: usize) -> Self {
        self.location = Some((line, col));
        self
    }
}
```

```rust
        let mut mappings = std::collections::HashMap::new();
        mappings.insert("i32".to_string(), "number".to_string());
        mappings.insert("i64".to_string(), "bigint".to_string());
```

### 4.4 代码格式化

```rust
/// 辅助函数：缩进代码
pub fn indent(code: &str, level: usize) -> String {
    let indent_str = "  ".repeat(level);
    code.lines()
        .map(|line| {
            if line.trim().is_empty() {
                String::new()
            } else {
                format!("{}{}", indent_str, line)
            }
        })
        .collect::<Vec<_>>()
        .join("\n")
}

/// 辅助函数：生成临时变量名
pub struct TempVarGenerator {
    counter: std::cell::RefCell<usize>,
}

impl TempVarGenerator {
    pub fn new() -> Self {
        Self {
            counter: std::cell::RefCell::new(0),
        }
    }
    
    pub fn next(&self) -> String {
        let mut counter = self.counter.borrow_mut();
        *counter += 1;
        format!("__temp_{}", counter)
    }
}
```

## 依赖关系

simple_x_cli 依赖以下内部组件，其变更会直接影响本项目：

- **x-lexer**：词法分析器
- **x-parser**：语法分析器
- **x-interpreter**：解释器

## 开发流程

### 常规开发流程

1. 修改 `src/main.rs`
2. 运行 `cargo run -- run <file.x>` 测试变更
3. 运行 `cargo test` 确保无错误
4. 提交更改

### 构建与运行

```bash
# 构建
cargo build

# 运行
cargo run -- run <file.x>

# 运行测试
cargo test

# 格式化代码
cargo fmt
```
